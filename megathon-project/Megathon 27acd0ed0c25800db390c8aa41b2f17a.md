- *Tier explanation :*
    
     üéüÔ∏è Sub Tiers in Twitch
    
    Tier 1 ‚Üí Base subscription (most common).
    
    Tier 2 ‚Üí Costs more, gives extra perks (usually counted as 2x Tier 1 value).
    
    Tier 3 ‚Üí Premium sub, often treated as 6x Tier 1 value.
    
    ‚è±Ô∏è How They Affect the Megathon Timer
    
    The widget usually has a conversion system where each sub = extra time added.
    
    But higher tiers/gifted subs add more time:
    
    Tier 1 sub / gift
    
    ‚Üí Adds the base amount of time (e.g., +1 min or +60 sec).
    
    Tier 2 sub / gift
    
    ‚Üí Counts as double a Tier 1, so it adds 2x time (e.g., +2 mins).
    
    Tier 3 sub / gift
    
    ‚Üí Counts as six Tier 1 subs, so it adds 6x time (e.g., +6 mins).
    
    üéÅ Gifted Subs
    
    Gift 1 Tier 1 ‚Üí +1 min.
    
    Gift 5 Tier 1s ‚Üí +5 mins.
    
    Gift 1 Tier 2 ‚Üí +2 mins.
    
    Gift 1 Tier 3 ‚Üí +6 mins.
    
    Gift 10 (mixed) ‚Üí Widget auto-calculates total time boost based on sub tiers.
    
    ‚ö° Example in Action
    
    Let‚Äôs say your Megathon is set to +1 min per Tier 1 sub:
    
    A Tier 1 gift adds +1 min.
    
    A Tier 2 sub adds +2 min.
    
    A Tier 3 sub adds +6 min.
    
    If someone gifts 5 Tier 1 subs ‚Üí +5 min to timer.
    
    If someone gifts 10 subs with 2 of them Tier 3s ‚Üí (8 √ó 1min) + (2 √ó 6min) = 20 mins added.                                                                             Upgrade Your Tier
    
    If you‚Äôre Tier 1, you can upgrade to Tier 2 or Tier 3.
    
    This counts as a new sub at the higher tier and adds more perks (and more timer boost in Megathon).
    
    Gift a Sub
    
    You can gift a sub to yourself indirectly via the ‚ÄúBuy a Sub for a Friend‚Äù system if you have another Twitch account.
    
    Or gift a sub to someone else on the channel.
    
    Resubscribe When Your Current Sub Expires
    
    Subscriptions are monthly. Once your current sub ends, you can resubscribe again.
    
    ‚ö° Key Notes
    
    Same-tier duplicate subscription is not allowed.
    
    Megathon widget behavior: It will only count the new tier/sub level for timer bonuses.
    
    Upgrade or gifting is the way to add more ‚Äúsub value‚Äù while already subscribed.
